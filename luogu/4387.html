    题目背景

    qwqqwqqwq函数最可爱！qwq

    题目描述

    付公主给定一个二次函数（给定方式见输入格式，接下来对于该函数进行n次操作）

    F a b意为将现在函数以（a，b）这个点为旋转中心，旋转$180^{\circ}$

    R a 意为将这个函数向右移动a格

    L a意为将这个函数向左移动a格

    T （x/y）a意为将这个函数以直线x/y=a翻折（x=a或者y=a，见输入是x或者y）

    J a代表将直线x=a左侧函数的所有点纵坐标改为它的绝对值

    在翻转以后，求直线y=k与改变之后的函数有m个交点时k的取值范围，如果不可能有m个交点，输出“Princess Fu is so cute!”

    输入输出格式

    输入格式：

    第一行，六个数，x1，y1，x2，y2，x3，y3，代表二次函数上任意三个点坐标（x1，y1），（x2，y2），（x3，y3）

    第二行，n

    第三～n+2行，n种操作

    第n+3行，一个数，m

    输出格式：

    输出取值范围，用区间表示。例如[2,4)或者(1,8)，无穷大用kkk表示，例如(-kkk,+kkk)，如果不可能，输出题目中要求的输出。

    答案请向下取整。若答案本是大于向下取整后的答案，请使用开区间。

    保证答案区间唯一

    输入输出样例

    输入样例#1： 复制 
    0 0 1 1 -1 1
2
R 3
J 2
2
    输出样例#1： 复制 
    (0,+kkk)

    说明

    对于30%的数据，$n \leqslant 100$

    对于60%的数据，$n \leqslant 5000$

    对于100%的数据，$n \leqslant 100000$，二次函数系数均为整数

    直接输出“Princess Fu is so cute!”，会发生不可言喻的事。
